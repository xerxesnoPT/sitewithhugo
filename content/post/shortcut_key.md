## Tmux

- ? : 列出所有帮助信息
- [ : 复制模式, vim操作模式，按下回车确定
- ] : 粘贴
- { : 切换pane
- , : 改变window的名字。
- . : 改变window的编号
- t : 显示时间
- alt+方向键: 调整pane大小.
- q : 显示面板编号


## spacevim 

- [ b: 跳至前一个buffer
- ] b: 后一个
- [ l: 跳至前一个错误。
- Tab: 窗口跳转
- <leader> + l p: 查看所有安装的插件
- spc + c + l: 快速注释
